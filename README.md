Features
--------------

- User authentication with persistent Todos
- Responsive MaterialUI design
- RESTful API
- Body / Params type checking


Getting Started
----------

1. Clone / Download the repository

1. Install dependencies

   ```bash
   npm install
   ```

1. Build the project

   ```bash
   npm run build
   ```

1. Run the project

   ```bash
   npm run dev
   ```

1. (Run only client)

   Navigate to `/frontend`
   ```bash
   npm install
   npm run dev
   ```
